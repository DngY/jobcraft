# jobcraft
An AI-powered cover letter generator built with Gradio and the Tongyi Qwen model from Alibaba. It allows users to upload their resumes (in .docx or .pdf format), input a job description, and specify a word limit. Leveraging the uploaded resume and job details, the tool generates a tailored cover letter that highlights the candidateâ€™s relevant skills and experience, all within the specified word count. Simple, efficient, and privacy-aware, this tool helps job seekers craft professional application letters effortlessly.
